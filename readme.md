# Tererê for Zen Browser

Light theme inspired by tererê — buttery cream paper, amber gold, and deep teal.

Part of the [Mate theme family](https://github.com/mateCreations) available for Neovim, VSCode, Obsidian, and Hyprland.

## Palette

| Role       | Color     | Preview |
|------------|-----------|---------|
| Background | `#fbf1c7` | Creme Amanteigado |
| Surface    | `#ebdfb0` | Shadow surface |
| Foreground | `#3c3836` | Cinza Carvão |
| Accent     | `#b57614` | Âmbar Dourado |
| Border     | `#928374` | Cinza Taupe |

## Installation

Install directly from the [Zen Browser theme marketplace](https://zen-browser.app/themes).
